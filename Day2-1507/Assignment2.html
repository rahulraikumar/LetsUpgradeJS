Explain with examples the remaining methods of String and Array



Below methods can extract string's part:

            
   1. slice(start, end) --> parameters are the indexes
  
   Example:
   let str= "I am a letsUpgrade student!!!";
   let slicedStr = str.slice(7,18); 
   let negSlcStr = str.slice(-3,-1);


   2. substring(start, end) -->Similar to slice()

   Example:
   let subString = str.substring(19,26); //substr doesnt take negative parameters

   
   3. substr(start, length) --> second parameter is the length of the extracted part

   Example:
   let subStr = str.substr(11, 7);

replace() will replace a part of the string

    replace(old part, new part)

    Example:
    let replacedStr = str.replace("student", "rowdy");
    console.log("replacedStr = ", replacedStr);

trim() --> it removes white spaces from both sides of a string

    let str1 = "      Hi people!!!     "
    let trimmedStr = str1.trim();
